Help
KTorrentTouch is an app for controlling a KTorrent program running on a seperate computer. This app can help manage the torrents that KTorrent is downloading/seeding. This app does not download anything to this device.
In order for this app to work you need to enable the "Web Interface" plugin inside the KTorrent App. Once the plugin is enabled, you can set the port, username and password under Settings --> Configure KTorrent.
If you have any problems you can try emailing the developer here.
